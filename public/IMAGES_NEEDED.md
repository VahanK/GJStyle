# Images Needed for GJ Style Website

Please add the following images to this `/public` folder:

## Required Images

### 1. hero-jewelry.jpg
- **Purpose**: Main hero section background
- **Recommended size**: 1920x1080px or larger (will be cropped responsively)
- **Content**: Stunning lifestyle/product photo showcasing gold-plated jewelry
- **Note**: Should be high quality (90% JPEG quality) as it's the first thing visitors see

### 2. workshop-background.jpg
- **Purpose**: "Our Story" section background
- **Recommended size**: 1920x1080px
- **Content**: Workshop/craftsmanship image showing Lebanese jewelry-making
- **Note**: Will have a dark overlay, so can be slightly darker

### 3. material-gold-plating.jpg
- **Purpose**: "Material & Quality" section
- **Recommended size**: 800x1000px (portrait orientation)
- **Content**: Close-up of gold-plated jewelry showing quality and finish
- **Note**: Should highlight the beauty of the gold-plating

### 4. contact-background.jpg
- **Purpose**: Contact/CTA section background
- **Recommended size**: 1920x1080px
- **Content**: Elegant jewelry display or abstract gold texture
- **Note**: Will have a dark gradient overlay

### 5. og-image.jpg
- **Purpose**: Social media sharing preview (OpenGraph)
- **Exact size**: 1200x630px
- **Content**: GJ Style branding with jewelry imagery
- **Note**: This appears when the site is shared on social media

### 6. favicon.ico
- **Purpose**: Browser tab icon
- **Size**: 32x32px or 16x16px
- **Content**: GJ Style logo or "GJ" initials

### 7. apple-touch-icon.png
- **Purpose**: iOS home screen icon
- **Size**: 180x180px
- **Content**: GJ Style logo

## Image Guidelines

- Use high-quality, professional photography
- Ensure all images are optimized for web (compressed)
- Avoid including any product SKUs or prices in images
- Images should reflect the luxury, quality brand positioning
- Consistent color grading across all images is recommended
